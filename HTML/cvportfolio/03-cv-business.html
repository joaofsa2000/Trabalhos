About me

An app developed whilst doing a practical work for a class. It was made in C# and it's purpose is to simulate the real life equivelant. It can receive input of numbers and stars according to the game rules, then, if given permission, stores the key to a file along with its date of register. The user can choose to use the key he just input or choose a key from a file with the keys that he already had stored, he may choose one of them to use. The programme will generate a prize key and compare to the user key, giving the numbers he got correct and say if he got any prize.
